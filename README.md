# SNAILO

Gra zręcznościowa, w której Gracz steruje ślimakiem, którego celem jest
zbieranie jabłek pojawiających się na planszy i zdobywanie jak największej liczby
punktów. Podczas rozgrywki należy unikać kolizji z przeszkodami.
 
Wraz ze wzrostem liczby zdobytych punktów tempo
rozgrywki stopniowo rośnie, co zwiększa poziom trudności i wymaga coraz szybszej
reakcji gracza

## Wprowadzenie Bota 
Implementacja bota, który będzie automatycznie sterował ślimakiem. Bot będzie podejmował decyzje w czasie rzeczywistym, dążąc do maksymalizacji liczby zdobytych punktów oraz unikania przeszkód.
